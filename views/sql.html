Brief History of SQL

SQL was founded by IBM in the 1970s for large databases. As the language progressed

people working for IBM began to implement more and more features into the language

such as multi table and multi user tables. As adoption increased the language became

standarized and other well known companies such as Oracle adopted SQL based products.

Now SQL is the probably the second most used language out there. Link.
